# Kısayol ve Komut Rehberi

Günlük kullanımda hız kazanmak için kısayollar. Detaylı versiyonu: `x.md`

## Windows Temel Kısayolları

| Kısayol | Açıklama |
|---------|----------|
| `Ctrl + C/V/X` | Kopyala/Yapıştır/Kes |
| `Ctrl + Z/Y` | Geri al/İleri al |
| `Alt + Tab` | Uygulamalar arası geçiş |
| `Win + D` | Masaüstüne geçiş |
| `Win + E` | Dosya Gezgini |
| `Win + L` | Bilgisayarı kilitle |
| `Win + R` | Çalıştır penceresini aç |
| `Win + I` | Ayarlar |
| `Ctrl + Shift + Esc` | Görev Yöneticisi |
| `Win + V` | Pano geçmişi |

## Tarayıcı Kısayolları

| Kısayol | Açıklama |
|---------|----------|
| `Ctrl + T` | Yeni sekme |
| `Ctrl + W` | Sekmeyi kapat |
| `Ctrl + Shift + T` | Kapalı sekmeyi aç |
| `Ctrl + L` | Adres çubuğuna git |
| `Ctrl + H` | Geçmiş |
| `Ctrl + D` | Yer imine ekle |

## VS Code Kısayolları

| Kısayol | Açıklama |
|---------|----------|
| `Ctrl + P` | Dosya ara |
| `Ctrl + Shift + P` | Komut paleti |
| `Ctrl + `` | Terminal aç/kapat |
| `Ctrl + /` | Satır yorumla |
| `Alt + Shift + ↓` | Satırı kopyala |
| `Ctrl + D` | Sonraki kelimeyi seç |

## Terminal (PowerShell/Bash)

| Komut | Açıklama |
|-------|----------|
| `cd klasör` | Dizin değiştir |
| `ls` | Dosya listele |
| `mkdir` | Klasör oluştur |
| `pwd` | Mevcut dizin |
| `clear` | Ekranı temizle |

## Geliştiriciler İçin

```bash
# Node.js projeleri
npm run dev      # Geliştirme sunucusu
npm test         # Testler
npm run build    # Derleme

# Git
git status       # Durum
git add .        # Hazırla
git commit -m "" # Commit
git push         # Push

# Docker
docker build -t adı .
docker run -p 3000:3000 adı
```
